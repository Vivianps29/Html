#  Virtual MakeUp

##  Nome
Vivian Petrille da Silva.

##  Descrição
Projeto de site de uma loja virtual de maquiagem desenvolvido com HTML e CSS.

##  Páginas
- index.html (Produtos)
- sobre.html (Sobre nós)
- contato.html (Contato)
- carrinho.html (Carrinho)

##  Funcionalidades
- Menu de navegação
- Produtos com imagens
- Vídeo do YouTube
- Links externos
- Página de informações sobre a loja
- Página de contato
